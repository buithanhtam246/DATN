## 🎯 API ENDPOINTS SUMMARY

### Create Review (POST)
```javascript
POST /api/reviews/create
{
  order_detail_id: 1,
  rating: 5,
  comment: "Sản phẩm rất tốt!"
}
→ 201 Created
```

### Get Product Reviews (GET)
```javascript
GET /api/reviews/product?variant_id=1&page=1&limit=10
→ 200 OK
{
  reviews: [...],
  stats: {
    avg_rating: 4.7,
    total_reviews: 25,
    rating_distribution: {...}
  }
}
```

### Get Order Reviews (GET)
```javascript
GET /api/reviews/order/1
→ 200 OK
[{...reviews...}]
```

### Update Review (PUT)
```javascript
PUT /api/reviews/update/1
{
  rating: 4,
  comment: "Updated comment"
}
→ 200 OK
```

### Delete Review (DELETE)
```javascript
DELETE /api/reviews/delete/1
→ 200 OK
```

## 📞 SUPPORT

### Having Issues?
1. Check: Error messages in console
2. Read: POSTMAN_TEST_GUIDE.md → Troubleshooting

```

   → Tip: Get token from login API first
```
